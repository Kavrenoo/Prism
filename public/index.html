Features

Everything in one hub

Built for power users who want speed, clarity, and control without the bloat.

>_
Command Terminal
Type commands directly. Rejoin, serverhop, goto players, adjust walkspeed and jumppower instantly.
🌐
Server Browser
Browse public servers filtered by ping, player count, or fewest players. Teleport with one click.
🎯
Prism Nametags
Custom floating tags for Prism users with profile pictures, clickable teleport, and distance LOD.
👥
Join Friends
See where other Prism users are across any game. Filter by friends, this game, or all games.
⚡
Live Stats
FPS and ping counters built into the hub with color-coded indicators for performance at a glance.
⚙️
Settings
Auto-execute toggles, custom sound IDs, volume sliders, and per-command auto-run preferences.
